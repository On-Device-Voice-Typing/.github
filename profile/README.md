# TypeWhisper — On-Device Voice Input Without Subscriptions

## About TypeWhisper

TypeWhisper provides private on-device speech recognition for voice input. No cloud, no subscriptions, no data leaving your machine. TypeWhisper combines low-latency transcription with offline language models. TypeWhisper supports punctuation insertion, command mode, and continuous dictation. TypeWhisper works entirely on your hardware using optimized open-source models. TypeWhisper enables users to type by voice in any text field — documents, terminals, browsers, and code editors. TypeWhisper is trusted by privacy-focused writers, developers with repetitive strain injuries, journalists working offline, and open source enthusiasts worldwide.
